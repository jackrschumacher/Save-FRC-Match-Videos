How do I get a free TBA API Key?
  1. Go to thebluealliance.com/account and log in.
  2. Scroll down to the Read API Keys section.
  3. Type a description (like "Video Extractor") into the text box and click Add New Key.
  4. Copy the long string of letters and numbers it generates and paste it into the field below.
